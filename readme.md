  # 網站 Sistema de Gerenciamento de Treinos
  ![Texto Alternativo para acessibilidade](/readmeicons/image.png)
  ![Texto Alternativo](/readmeicons/Captura%20de%20tela%202026-06-14%20163654.png)


Projeto desenvolvido para auxiliar no cadastro e organização de treinos físicos de forma simples e intuitiva.

##   功能性 Funcionalidades

* Cadastro de exercícios
* Definição de séries e repetições
* Organização dos exercícios por treino
* Interface responsiva
* Integração com API para armazenamento dos dados

## 實施 Como executar o projeto

### 1. Clone o repositório

```bash
git clone https://github.com/Valakzz/TrabalhoWebFinalAcademy/tree/main
```

### 2. Acesse a pasta

```bash
cd nome-do-projeto
```

### 3. Instale as dependências

```bash
npm install
```

### 4. Inicie o servidor

```bash
npm start
```

### 5. Abra o projeto

Abra o arquivo `index.html` ou acesse a porta configurada no navegador.



## 客觀的 Objetivo

O objetivo deste projeto é facilitar o gerenciamento de treinos, permitindo o cadastro e a visualização de exercícios de forma organizada e eficiente.
